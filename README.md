## 📊 Design System

**Aesthetic Direction:** Technical blueprint with strict 12-column grid layouts, heavy 900-weight typography for metrics, hairline 1px borders enclosing all content blocks, left-aligned flush text with generous negative space, monochromatic base with single vibrant orange accent.

**Color Palette:**
- Background: `#0A192F` (Deep Navy)
- Title Text: `#E6F1FF` (Light Blue)
- Body Text: `#8892B0` (Muted Blue)
- Accent Color: `#FF8C00` (Vibrant Orange)
- Border Color: `#112240` (Dark Blue)

**Typography:**
- **Headlines:** Space Grotesk 900 (64px Front, 32px Content)
- **Body Text:** DM Sans 400 (20px Front, 16px Content)
- **Data Labels:** JetBrains Mono 700 (20px Front, 12px Content)

**Visual Elements:**
- Hairline 1px borders around all content blocks
- Left-aligned, flush text layout
- Generous negative space between sections
- Heavy typography for emphasis on metrics and numbers
- Orange accent used sparingly for key data points and CTAs
